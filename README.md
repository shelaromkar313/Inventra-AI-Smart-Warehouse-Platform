# Smart Warehouse AI Platform

A production-grade multi-service AI platform for warehouse management.

## Tech Stack
- **Core Backend**: Node.js & Express.js
- **Database**: PostgreSQL
- **Frontend**: React (Vite)
- **AI Services (Python)**: 
    - Demand Forecasting (ML)
    - Computer Vision (CV)
    - RAG Assistant

## Project Structure
- `/server`: Node.js Express Backend
- `/ml-service`: Python Machine Learning service (XGBoost, Regression)
- `/cv-service`: Python Computer Vision service (OpenCV, YOLO)
- `/client`: React Frontend (TBD)
